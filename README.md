## 🎯 Problem Statement

Global geopolitical dynamics are extremely complex and difficult to understand or simulate in real-time. There is no simple system where users can visualize how information warfare and infrastructure disruption affect global stability.

---

## 💡 Solution

We built **Empire of Shadows**, an AI-powered geopolitical simulation where countries act as intelligent agents.

Users act as **Shadow Operators**, influencing the world through:

* 🗣️ **Whispers** → Secret information to manipulate decisions
* 💥 **Attacks** → Direct disruption of infrastructure

This creates a dynamic system where AI agents react, adapt, and form evolving global narratives.

---

## 🧩 Key Features

* 🤖 AI Agents representing countries
* 🧠 Autonomous decision-making using LLMs
* 🗣️ Whisper System (covert influence)
* 💥 Attack System (kinetic disruption)
* 🌍 Real-time Tactical Map (nodes & factions)
* 📜 Operation Ledger (logs every action)
* 🔁 Chain Reaction Engine (global impact simulation)

---

## 🏗️ Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Python (FastAPI)
* **AI Engine:** LangChain, Groq (Llama 3), Google Gemini
* **Database:** SQLite, Neo4j

## ⚙️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/priy0767/Empire-of-Shadows.git
cd Empire-of-Shadows
```

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file and add:

```
GROQ_API_KEY=your_key_here
GEMINI_API_KEY=your_key_here
```

Run backend:

```bash
uvicorn main:app --reload
```

### 3. Frontend Setup

```bash
cd ../frontend
npm install --legacy-peer-deps
npm run dev
```

## 🧠 How It Works

1. **AI Initialization:** Each country is an intelligent agent with its own strategy and faction.
2. **Agentic Engine:** Every cycle, agents communicate automatically using "whispers".
3. **User Control:** Players inject information using whispers or perform attacks.
4. **Reactions:** AI updates its decisions based on new inputs.
5. **Chain Reactions:** Actions spread across the global network, changing alliances and tensions.
